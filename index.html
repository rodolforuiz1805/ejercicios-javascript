Ejercicio 1

Completa la función para imprimir la cadena: "This is a sentence"

Ejercicio 2

Escribe una función que:

  • Reciba un arreglo de números.
  • Doble el valor de cada número en el arreglo. Ejemplo: Dado un arreglo [1, 2, 4, 5]. La salida debería ser [2, 4, 8, 10]
  • Imprima el nuevo arreglo actualizado.

Ejercicio 3

Escribe un programa para calcular la suma y el factorial (multiplicación) de un arreglo de números. Imprime la suma y el producto. Ejemplo: Dado un arreglo [1, 2, 3, 4] La suma es 10. El producto es 24.

Ejercicio 4

Crea un programa que recorra los 2 arreglos; si hay cursos comunes, imprímelos en la consola.

Ejercicio 5

Escribe un comando que:

  • Escribe el comando para remover "Dani" del arreglo.
  • Escribe el comando para remover "Dani" del arreglo.
  • Escribe el comando para remover "Juan" del arreglo.
  • Escribe el comando para mover "Luis" al principio del arreglo.
  • Escribe el comando para agregar tu nombre al final del arreglo.
  • Usando un bucle, itera a través de este arreglo y después de console.log "Maria", sal del bucle.
  • Escribe el comando que da el índice donde se encuentra "Maria".
  • Al final del ejercicio, debería haber 4 personas en el arreglo.
Ejercicio 6

Escribe una función que realice el algoritmo de burbuja.

  • Entrada [3, 6, 12, 5, 100, 1]
  • Salida [1, 3, 5, 6, 12, 100]
